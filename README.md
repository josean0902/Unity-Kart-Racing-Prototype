# Unity Kart Racing Prototype

## Versión de Unity

**2022.3.62f2**

---

## Cómo ejecutar

1. Abrir el proyecto con Unity **2022.3.62f2**.
2. Abrir la escena **UnityKartRacingPrototype**.
3. Pulsar **Play**.

---

## Controles

* **Mantener clic izquierdo:** iniciar y mantener el derrape.
* **Soltar clic izquierdo:** activar el boost (si se ha acumulado suficiente carga durante el derrape).

---

## Funcionalidades implementadas

* Movimiento automático del vehículo.
* Sistema de derrape.
* Sistema de boost basado en la duración del derrape.
* Efectos visuales de chispas.
* Ajustes de game feel para una conducción más fluida y responsiva.
* Circuito completamente jugable.
* Sistema de vueltas.
* Cronómetro de carrera.
* Sistema de ranking.
* Coleccionables (monedas).
* Efectos de sonido.
* Cámara de seguimiento.
* IA básica de oponentes mediante seguimiento de waypoints.
