# 🇬🇧 English Version

## Unity Kart Racing Prototype

### Unity Version
2022.3.62f2

---

### How to Run

- Open the project with Unity 2022.3.62f2.  
- Open the scene UnityKartRacingPrototype.  
- Press Play.

---

### Controls

- **Hold left mouse button:** start and maintain drifting.  
- **Release left mouse button:** activate boost (if enough charge has been accumulated during the drift).

---

### Implemented Features

- Automatic vehicle movement  
- Drifting system  
- Boost system based on drift duration  
- Spark visual effects  
- Game feel tuning for smoother and more responsive driving  
- Fully playable track  
- Lap system  
- Race timer  
- Ranking system  
- Collectibles (coins)  
- Sound effects  
- Follow camera  
- Basic AI opponents using waypoint tracking

# 🇪🇸 Versión en Español

## Prototipo de Carreras Unity Kart

### Versión de Unity
2022.3.62f2

---

### Cómo ejecutar

- Abrir el proyecto con Unity 2022.3.62f2.  
- Abrir la escena UnityKartRacingPrototype.  
- Pulsar Play.

---

### Controles

- **Mantener clic izquierdo:** iniciar y mantener el derrape.
